Модуль: "Площади геометрических фигур на плоскости"

1. Назначение модуля
Данный модуль предназначен для вычисления площадей геометрических фигур на плоскости. Модуль содержит набор функций для вычисления площадей фигур по координатам вершин или заданным параметрам.

2. Состав модуля
Модуль включает следующие файлы:
Header.h — заголовочный файл с объявлениями функций;
Header.cpp — файл реализации функций;
IDZ_8_1_Pavlov.cpp — тестирующая программа.

3. Подключение модуля
Для использования модуля необходимо подключить заголовочный файл:
#include "Header.h"

4. Описание функций
squareTriangle(...)
Назначение:
Вычисление площади треугольника по координатам трёх вершин.
Формат:
double squareTriangle(double x1, double y1, double x2, double y2, double x3, double y3);

Параметры:
x1, y1 — координаты первой вершины;
x2, y2 — координаты второй вершины;
x3, y3 — координаты третьей вершины.

Возвращаемое значение:
Площадь треугольника.

squareRectangle(...)
Назначение:
Вычисление площади прямоугольника.
Формат:
double squareRectangle(double x1, double y1, double x2, double y2, double x3, double y3, double x4, double y4);
Параметры:
Координаты четырёх вершин прямоугольника.
Возвращаемое значение:
Площадь прямоугольника.

squareParallelogram(...)
Назначение:
Вычисление площади параллелограмма.
Формат:
double squareParallelogram(double x1, double y1, double x2, double y2,double x3, double y3, double x4, double y4);
Параметры:
Координаты четырёх вершин параллелограмма.
Возвращаемое значение:
Площадь параллелограмма.

squareRhomb(...)
Назначение:
Вычисление площади ромба.
Формат:
double squareRhomb(double x1, double y1, double x2, double y2, double x3, double y3, double x4, double y4);
Параметры:
Координаты вершин ромба.
Возвращаемое значение:
Площадь ромба.

squareTrapeze(...)
Назначение:
Вычисление площади трапеции.
Формат:
double squareTrapeze(double x1, double y1, double x2, double y2, double x3, double y3, double x4, double y4);
Параметры:
Координаты вершин трапеции.
Возвращаемое значение:
Площадь трапеции.

squareCircle(double r)
Назначение:
Вычисление площади круга по радиусу.
Формат:
double squareCircle(double r);
Параметры:
r — радиус круга.
Возвращаемое значение:
Площадь круга.

squareCircleSector(double r, double alpha)
Назначение:
Вычисление площади сектора круга.
Формат:
double squareCircleSector(double r, double alpha);
Параметры:
r — радиус круга;
alpha — угол сектора в градусах.
Возвращаемое значение:
Площадь сектора круга.

5. Принцип работы модуля
Пользователь выбирает необходимую фигуру, после чего вводит координаты вершин фигуры или необходимые параметры. Программа вызывает соответствующую функцию модуля и выводит вычисленную площадь фигуры.
